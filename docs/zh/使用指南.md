## 公共功能

### 博客

- 访问 `/blog` 浏览全部文章
- 通过搜索框按标题或内容搜索
- 按日期范围筛选
- 点击文章标题阅读全文
- 每篇文章包含：标题、日期、摘要、标签、markdown 正文

### 文件（图片）

- 访问 `/files` 查看公开图片
- 按相册分类组织
- 图片通过 `/api/uploads/<filename>` 提供，带有长期缓存

### 工具（`/tools`）

登录后可见五个 tab：

- **待办** — 任务列表，支持"已完成/未完成"筛选
- **日记** — 按日期记录，支持 markdown
- **命理** — 中国传统命理工具（八字、紫微、六爻、梅花易数）
- **AI 对话** — 与已配置的 AI 服务商聊天
- **订阅** — AI 生成的订阅源摘要

## 管理后台

登录密码后访问 `/admin/*` 进入管理面板。

| 页面 | 用途 |
|------|------|
| `/admin/blog` | 创建、编辑、删除博客文章（带 markdown 编辑器）|
| `/admin/diary` | 管理日记 |
| `/admin/tools` | 管理待办列表 |
| `/admin/files` | 上传图片，组织相册 |
| `/admin/skills` | 编辑 AI 技能（system prompt、模板）|
| `/admin/ai-config` | 配置 AI 服务商（OpenAI、Anthropic）|
| `/admin/subscriptions` | 添加、编辑、删除订阅源 |
| `/admin/x-post` | 通过 AI 生成推文并发布到 X/Twitter |

## AI 对话

1. 前往 **管理后台 → AI 配置**
2. 点击 **+ 新增服务商**
3. 填写：
   - **名称** — 如"Claude"
   - **API 类型** — `openai` 或 `anthropic`
   - **API URL** — 服务基础 URL（如 `https://api.anthropic.com`）
   - **API Key** — 密钥
   - **模型** — 如 `claude-opus-4-6` 或 `gpt-4o`
4. 至少将一个服务商勾选为 **默认**
5. 点击 **测试连接** 验证
6. 点击 **保存**

然后前往 **工具 → AI 对话**，从下拉框选择服务商开始聊天。

## 订阅系统

追踪博客、GitHub 仓库、X 账号等源的最新动态。

### 添加订阅源

1. 前往 **管理后台 → 订阅**
2. 点击 **+ 新增源**
3. 填写：
   - **名称** — 显示名
   - **URL** — 源地址
   - **分类** — `github`、`x`、`selfblog`、`rss`、`newsletter`、`reddit` 或 `other`
   - **抓取间隔** — 自动抓取间隔秒数（默认 3600 = 1 小时）
4. 点击 **保存**

### 查看摘要

1. 前往 **工具 → 订阅**
2. 点击 **全部刷新** 抓取并生成最新摘要
3. AI 生成的摘要包含：
   - **最新动态** — 核心概述
   - **重点更新** — 3-6 条按重要性排序
   - **值得关注** — 趋势洞察

每条摘要由 `subscription` 技能生成，使用默认 AI 服务商。

## 发布到 X

将博客或日记转为推文并发布。

1. 前往 **管理后台 → Post to X**
2. 选择来源：**博客**、**日记** 或 **自定义**
3. 选中一条（或粘贴自定义内容）
4. 点击 **🤖 用 AI 生成推文** — 生成单条推文或推文串
5. 根据需要编辑（字数会实时显示）
6. 附加图片：
   - **📁 从本地电脑** — 选择本地文件
   - **🖼️ 从站点文件** — 从已上传的图片中选
7. 点击 **🐦 发布到 X**

### 前置条件

- `.env.local` 中配置 `X_CONSUMER_KEY`、`X_CONSUMER_SECRET`、`X_ACCESS_TOKEN`、`X_ACCESS_TOKEN_SECRET`
- X App 权限必须为 **Read and Write**

## 命理工具

**工具 → 命理** 包含四种占卜方式：

- **八字** — 根据生辰进行四柱分析
- **紫微斗数** — 紫微星盘
- **周易六爻** — 卦象占卜
- **梅花易数** — 基于数字的快速占卜

每种方式使用专门的 AI 技能进行详细解读。
